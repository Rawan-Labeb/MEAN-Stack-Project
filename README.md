# E-Commerce & Inventory Management System

## 📌 Overview
This system is designed to streamline operations for both **online shopping** and **offline branch activities**. It provides a seamless way to manage stock levels, track orders, and process payments, ensuring efficient coordination between online and offline workflows.

## 🚀 Key Features

### 🔑 Multi-Role User System
- **Admin Dashboard**: Complete store management.
- **Sales Clerk Portal**: Branch management interface.
- **Seller Dashboard**: Vendor product management.
- **Cashier Interface**: In-store purchases processing.
- **Customer Experience**: Smooth shopping and order tracking.

### 🏢 Branch Management
- Supports both **online and offline branches**.
- **Inventory distribution** between branches.
- **Branch-specific sales analytics**.
- **Branch cashier management**.

### 📦 Inventory System
- **Main inventory control** with stock tracking.
- **Sub-inventory tracking** for each branch.
- **Product distribution request system**.
- **Stock level monitoring** to prevent shortages.

### 📊 Sales & Analytics
- **Comprehensive sales dashboards** with interactive charts.
- **Monthly performance tracking** per branch.
- **Best-selling product analytics**.
- **Comparison of online vs offline sales**.

### 🛒 Customer Experience
- **Responsive product catalog** with advanced search & filters.
- **User profile management**.
- **Order history tracking**.
- **Complaint submission system**.

### 📦 Order Processing
- **Online shopping cart system**.
- **Offline order processing**.
- **Real-time order status tracking**.
- **Multi-branch order fulfillment**.

## 🛠️ Tech Stack

### ✅ Frontend
- **Angular (v19)**
- **Angular Material, PrimeNG, ngx-charts** for a dynamic UI.

### ✅ Backend
- **Node.js & Express** for scalable API design.

### ✅ Database
- **MongoDB** for flexible data storage.
- **Mongoose** for schema management.

### ✅ Authentication & Security
- **JSON Web Tokens (JWT)** for secure authentication.
- **bcryptjs** for password hashing.

### ✅ Additional Features
- **Stripe** for payment processing.
- **Nodemailer** for email notifications.
- **ImageKit** for efficient image handling.

## 🚀 How to Run the Project

### 1️⃣ Clone the Repository
```sh
git clone https://github.com/MEAN-Stack-Project
cd MEAN-Stack-Project
```

### 2️⃣ Backend Setup
```sh
cd backend
npm install
npm start
```

### 3️⃣ Frontend Setup
```sh
cd frontend
npm install
ng serve
```

### 4️⃣ Access the Application
- Open **http://localhost:4200/** in your browser.
- The backend runs on **http://localhost:5000/** by default.
